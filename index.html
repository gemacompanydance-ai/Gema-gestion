⚙ Conectar Google Sheets

Pegá la URL de tu Web App de Google Apps Script para conectar la app con tu planilla.

1
Abrí tu Google Sheet → Extensiones → Apps Script
2
Pegá el código del archivo Code.gs y guardá
3
Corré la función setupInicial() una vez
4
Hacé clic en Implementar → Nueva implementación → Tipo: App web → Acceso: Cualquier usuario → Copiar URL
5
Pegá la URL acá abajo y hacé clic en Guardar y conectar
